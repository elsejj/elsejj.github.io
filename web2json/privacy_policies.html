Web2Json Privacy Policies

How this extension handle your data

To use this extension, you need to provide some LLM service information, this information is stored in your extension's sync store, which is provided by Google Chrome, and synchronized across your devices., this information is protected by Google Chrome's security policies, and only the extension can access it.

The configuration of each web url is stored in your extension's local store, which is also provided by Google Chrome, and only the extension can access it.

When you do extraction, the extension will send the information to the LLM service, and the service will return the extracted data, then the extension will save the data in your extension's local store, which is also provided by Google Chrome, and only the extension can access it.

Because the LLM service is provided by yourself, so you need to check the service's privacy policies to know how the service handle your data.

All the data is stored in your local device, and not shared with any other parties except the LLM service you provided.

Since the plugin's only external access point is the large language model service you provide, it has no opportunity to collect any of your usage data.

How to remove your data

When you want remove your data, you can click the "Clear Data" button in the extension's settings page, then the extension will remove all the data in your extension's local store.

Or you can just uninstall this extension, Chrome will remove all the data of this extension
